# JavaScript Calculator

A simple calculator built using HTML, CSS, and JavaScript.  
It performs basic arithmetic operations and supports both button clicks and keyboard input.

## Features
- Addition, subtraction, multiplication, and division
- Clear (C) and Clear Entry (CE) functions
- Backspace support
- Decimal input
- Keyboard input support
- Responsive calculator UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the repository
2. Open the `index.html` file in your browser

## File Structure
```text

How It Works
The user clicks buttons or uses the keyboard to enter numbers
JavaScript handles the input and performs calculations
The result is displayed on the screen
Example Operations

9 × 5 = 45
10 ÷ 2 = 5

Prevents division by zero
Supports keyboard keys like:

+, -, *, /
